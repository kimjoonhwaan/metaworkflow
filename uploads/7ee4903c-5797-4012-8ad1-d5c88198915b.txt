워크플로우 생성 가이드

이 문서는 AI 워크플로우 생성 시스템의 사용법을 설명합니다.

1. 기본 워크플로우 구조
- 워크플로우는 여러 스텝으로 구성됩니다
- 각 스텝은 특정 작업을 수행합니다
- 스텝 간에는 데이터가 전달됩니다

2. 지원되는 스텝 타입
- API_CALL: 외부 API 호출
- PYTHON_SCRIPT: Python 코드 실행
- LLM_CALL: AI 모델 호출
- CONDITION: 조건부 실행
- DELAY: 대기 시간
- NOTIFICATION: 알림 전송
- DATA_TRANSFORM: 데이터 변환

3. 워크플로우 생성 팁
- 명확하고 구체적인 요구사항을 제시하세요
- 입력 데이터 형식을 명시하세요
- 출력 결과 형식을 지정하세요
- 에러 처리 방식을 고려하세요

4. Python 스크립트 작성 규칙
- --variables JSON 형식으로 입력 받기
- stdout으로 최종 JSON 결과 출력
- stderr로 디버그/로그 메시지 출력
- 예외 처리를 포함하세요

5. 일반적인 워크플로우 예시
- 데이터 수집 → 처리 → 저장
- API 호출 → 데이터 변환 → 알림
- 조건부 실행 → 분기 처리 → 결과 통합

이 가이드를 참고하여 더 정확하고 효율적인 워크플로우를 생성하세요.
